# Fifa-21-Cleaning-and- Data Storytelling
This dataset analyzes FIFA player attributes to solve three business problems: 1) The correlation between Work Rates (A/W &amp; D/W) and market value; 2) Whether left-foot dominance offers a skill advantage over right-footed players; and 3) The relationship between age, composure, and discipline to determine if veterans improve team sportsmanship.
